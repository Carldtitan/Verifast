Design a synchronous Moore finite state machine named 'fsm_idle'.
Clock 'clk' and synchronous active-high reset 'rst' are implicit inputs.
Inputs: i0 (1-bit input), i1 (1-bit input).
Outputs: o0 (2-bit output).
On reset the machine is in state S0.
It has 2 states: S0, S1.

Per-state behavior (Moore — outputs depend on the current state only):
- In S0: outputs o0=2.
    if i1 is 1, go to S1 (checked in this order);
    otherwise go to S0.
- In S1: outputs o0=3.
    otherwise go to S0.

Write a single synthesizable SystemVerilog module with this exact interface. Output only the module.
