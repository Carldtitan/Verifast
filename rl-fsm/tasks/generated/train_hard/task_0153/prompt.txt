Design a synchronous Moore finite state machine named 'fsm_lock'.
Clock 'clk' and synchronous active-high reset 'rst' are implicit inputs.
Inputs: i0 (1-bit input), i1 (1-bit input).
Outputs: o0 (1-bit output).
On reset the machine is in state S0.
It has 5 states: S0, S1, S2, S3, S4.

Per-state behavior (Moore — outputs depend on the current state only):
- In S0: outputs o0=0.
    otherwise go to S3.
- In S1: outputs o0=0.
    otherwise go to S3.
- In S2: outputs o0=0.
    if i0 is 1, go to S0 (checked in this order);
    otherwise go to S3.
- In S3: outputs o0=0.
    if i0 is 1, go to S4 (checked in this order);
    if i1 is 1, go to S0 (checked in this order);
    otherwise go to S1.
- In S4: outputs o0=1.
    otherwise go to S1.

Write a single synthesizable SystemVerilog module with this exact interface. Output only the module.
